This project uses a trained Artificial Neural Network (ANN) to predict whether a customer is likely to churn based on customer profile and account information.

The application is built with:
- TensorFlow / Keras
- Scikit-Learn
- Pandas
- NumPy
- Streamlit

Features:
- Real-time churn prediction
- Probability score visualization
- Customer feature preprocessing
- One-Hot Encoding and Label Encoding
- Standardized feature scaling
- Interactive Streamlit web interface
